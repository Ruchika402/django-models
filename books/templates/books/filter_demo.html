🐍 Django ORM - filter() & exclude() Demo

📊 Total Books in Database: {{ total_books }}
1️⃣ filter() - Basic Filtering
fiction_books = Book.objects.filter(genre='FIC')
✅ Result: {{ fiction_books.count() }} Fiction books found {% for book in fiction_books|slice:":5" %}
📘 {{ book.title }} by {{ book.author }}
{% endfor %}
2️⃣ exclude() - Excluding Records
recent_books = Book.objects.exclude(published_date__year__lt=2000)
✅ Result: {{ recent_books.count() }} books published in/after 2000 {% for book in recent_books|slice:":5" %}
📘 {{ book.title }} ({{ book.published_date.year }})
{% endfor %}
3️⃣ Chain Filters - AND Condition
popular_fiction = Book.objects.filter(genre='FIC').filter(rating__gte=4)
✅ Result: {{ popular_fiction.count() }} popular fiction books (4+⭐) {% for book in popular_fiction %}
📘 {{ book.title }} - Rating: {{ book.rating }}⭐
{% endfor %}
4️⃣ Multiple Conditions (Comma Separated)
classic_sci_fi = Book.objects.filter(genre='SCI', published_date__year__lt=2000)
✅ Result: {{ classic_sci_fi.count() }} classic Sci-Fi books (before 2000) {% for book in classic_sci_fi %}
📘 {{ book.title }} ({{ book.published_date.year }})
{% endfor %}
5️⃣ Chaining exclude()
not_fiction_not_low_rating = Book.objects.exclude(genre='FIC').exclude(rating__lt=3)
✅ Result: {{ not_fiction_not_low_rating.count() }} non-fiction books with rating ≥3 {% for book in not_fiction_not_low_rating|slice:":5" %}
📘 {{ book.title }} ({{ book.get_genre_display }}) - {{ book.rating }}⭐
{% endfor %}
6️⃣ __in Lookup (Multiple Values)
favorite_genres = Book.objects.filter(genre__in=['FIC', 'SCI', 'FAN'])
✅ Result: {{ favorite_genres.count() }} books in Fiction, Sci-Fi, or Fantasy {% for book in favorite_genres|slice:":5" %}
📘 {{ book.title }} ({{ book.get_genre_display }})
{% endfor %}
7️⃣ filter().exclude() - Combining Both
filtered = Book.objects.filter(rating__gte=3).exclude(published_date__year__lt=1990)
✅ Result: {{ filtered.count() }} books with rating ≥3 AND published after 1989 {% for book in filtered|slice:":5" %}
📘 {{ book.title }} ({{ book.rating }}⭐, {{ book.published_date.year }})
{% endfor %}
